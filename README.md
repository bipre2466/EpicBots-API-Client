<br />

## Installation

1. Cloner le repo
   ```sh
   git clone https://github.com/bipre2466/EpicBots-API-Client.git
   ```
2. Installer les packages NPM
   ```sh
   npm i
   ```
3. Exécuter le fichier `index.js`
   ```sh
   node index.js
   ```
